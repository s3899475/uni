Types of IPv6 address
* Unicast - An identifier for a single interface
* Anycast - IPv4 uses broadcast address. Identifier for a set on interfaces. Delivered to one interface, the nearest one, according to the routing protocol's measure of distance
* Multicast - Identifier for a set of interfaces. Packet sent us delivered to all interfaces
